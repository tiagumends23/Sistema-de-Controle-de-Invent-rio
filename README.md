# Sistema de Controle de Inventário em Java 📦☕

## 📝 Sobre o Projeto
Este é um projeto prático desenvolvido para consolidar e aprofundar os conceitos de **Programação Orientada a Objetos (POO)** na linguagem Java. O sistema simula o gerenciamento de um estoque de eletrônicos, permitindo o registro de diferentes tipos de produtos e o cálculo automático do capital investido no inventário.

## 🚀 Funcionalidades
- Cadastro de produtos genéricos e especializados (Smartphones e Smartwatches).
- Exibição de relatórios detalhados de estoque, respeitando as características únicas de cada tipo de produto.
- Cálculo dinâmico do valor total do inventário com base no preço e quantidade em estoque.
- *(Em breve)* Sistema de aplicação de cupons de desconto dinâmicos.

## 🧠 Conceitos e Tecnologias Aplicadas
Durante o desenvolvimento deste sistema, foram aplicados os seguintes pilares da POO e recursos do Java:
- **Encapsulamento:** Proteção de dados utilizando modificadores `private` e métodos de acesso (`getters` e `setters`).
- **Herança:** Criação de uma classe base (`Produto`) e classes filhas (`Smartphone` e `Smartwatch`) utilizando `extends`.
- **Polimorfismo:** Sobrescrita de métodos com `@Override` para comportamentos específicos de cada produto na hora de exibir os detalhes.
- **Estruturas de Dados:** Uso de `List` e `ArrayList` para manipulação dinâmica do estoque.
- **Boas Práticas:** Utilização da palavra-chave `final` para imutabilidade de atributos constantes.

## 🛠️ Como Executar
1. Certifique-se de ter o [Java JDK](https://www.oracle.com/java/technologies/downloads/) instalado na sua máquina.
2. Clone este repositório ou faça o download dos arquivos `.java`.
3. Abra a pasta do projeto em sua IDE favorita (VS Code, Eclipse, IntelliJ, etc.).
4. Execute o arquivo `Main.java` para visualizar o relatório de estoque no console.
